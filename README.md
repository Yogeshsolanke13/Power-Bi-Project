#  Diwali Sales Analysis Dashboard

##  Project Overview
This project analyzes **Diwali Festival Sales Data** to understand customer purchasing behavior. By analyzing over **28,000+ orders**, this interactive Power BI dashboard helps businesses identify top-selling products and best-performing regions to optimize sales and marketing strategies.

##  Key Insights (What I Found)
*   **Top Buyers:** Female customers made significantly more purchases than male customers.
*   **Target Age:** The most active buyers belong to the **26-35 years** age bracket.
*   **Top States:** Uttar Pradesh, Maharashtra, and Karnataka generated the highest sales revenue.
*   **Best Products:** Clothing & Apparel, Food, and Electronics emerged as the top-selling categories.

---

##  Tech Stack & Tools Used
*   **Business Intelligence:** Microsoft Power BI Desktop
*   **Data Source:** Diwali Sales Dataset (CSV format)
*   **Data Cleaning:** Power Query & Microsoft Excel (Handled empty cells/null values and fixed data types)
*   **Interactivity:** Dynamic Slicers (Filters) for Zones, States, and Occupations
